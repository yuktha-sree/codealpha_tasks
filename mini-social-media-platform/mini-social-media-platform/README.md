# Mini Social Media Platform

A complete internship-ready mini social media web app.

## Features

- User registration and login
- JWT authentication
- User profile
- Edit profile
- Create posts
- View all posts
- Like/unlike posts
- Add comments
- Follow/unfollow users
- Search users
- JSON file database for users, posts, comments, likes and followers
- Responsive UI

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: JSON file database
- Authentication: JWT + bcrypt password hashing

## How to Run

Open this folder in VS Code.

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm start
```

Open in browser:

```text
http://localhost:5000
```

## Test Flow

1. Register user 1.
2. Open another browser tab/incognito window.
3. Register user 2.
4. Create posts.
5. Like and comment on posts.
6. Search people and follow users.

## Project Description

Mini Social Media Platform is a full-stack web application that allows users to register, login, create profiles, publish posts, add comments, like posts and follow other users. The project uses HTML, CSS and JavaScript for the frontend, Express.js for the backend and a JSON file database for storing users, posts, comments, likes and followers.
